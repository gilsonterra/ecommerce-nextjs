## Tecnológias Utilizadas

- React
- Typescript
- NextJs
- Redux
- Vercel
- SSH - Serve Side Render
- Fetch (para consumir API)

## Getting Started

Para iniciar o servidor local execute:

```bash
yarn dev
```

Depos abra o endereço [http://localhost:3000](http://localhost:3000).
